Mobile Menu Test Page

Testing Instructions:

  1. Resize your browser window to mobile size (375px width or less)
  2. You should see a hamburger menu button (three lines) in the top right
  3. Click/tap the hamburger button
  4. The menu should slide down from the top
  5. Open browser console (F12) to see debug logs

What to check in console:

  • "Navbar Toggle: [object]" - confirms button found
  • "Navbar Menu: [object]" - confirms menu found
  • "Toggle button clicked!" - confirms click registered
  • "Toggle active: true" - confirms active class added
